Java Enterprise Online Project 
 ===============================
 
 Restaurant voter application was developed as a Spring/JPA/Rest app with authorization and authentication based on Roles. 
 Main technologies used:
 - Maven
 - Spring MVC, Security
 - JPA (Hibernate)
 - REST (Jackson)
 - Java 8 Stream and Time API
 - HSQL
